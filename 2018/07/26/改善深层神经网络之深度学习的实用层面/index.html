改善深层神经网络之深度学习的实用层面

发表于 2018-07-26 |

Settiing up your ML application

Train/dev/test sets
应用型机器学习是一个高度的迭代过程

训练神经网络时通常要做出很多决策：
神经网络分多少层
每层有多少个隐藏单元
学习速率是多少
各层采用哪种激活函数

为了找到更好的神经网络需要不断迭代更新自己的方案

数据集通常被分为数据集 验证集 测试集

数据集规模较小时，适用传统的划分比例
数据集规模较大时，验证集和测试集可以占到数据总量的10%以下

要确保训练和测试集来自同一分布

Bias and Variance


欠拟合，适度拟合和过拟合

可以通过几个指标来研究偏差和方差
理解偏差和方差两个关键指标数据是训练集误差和验证集误差
训练集误差小，验证集误差大：过拟合，高方差
训练集误差大，验证集误差大：欠拟合，高偏差
训练集误差大，验证集误差更大：高方差高偏差
最优误差/贝叶斯误差：如果最优误差很高，训练集误差验证集误差大也很正常

Basic recipe for machine learning

高偏差：规模更大的网络结构，延长训练时间
高方差：采集更多数据，正则化减少过拟合
