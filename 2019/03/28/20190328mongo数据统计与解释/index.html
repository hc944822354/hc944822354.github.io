mongo数据统计与解释

统计时间：2019-3-28

各个库的表及大小

本次数据总大小为681.7GB,共分为以下几个库：

  • db_name 大小76.4GB

    • IMP 大小 75GB
  • filesData 大小为39.4GB
    • IMP 大小 39GB
  • fileData_files大小为 98.94GB
    • IMP.files 大小3.49KB 测试数据，不用考虑
    • IMP.chunks 大小488KB 测试数据，不用考虑
    • FB2.files 大小8GB
    • FB2.chunks 90GB
  • ND_1大小为 6.077GB
    • file 6GB 可以不考虑
  • ND_1_files 大小为 461.738GB
    • 由于下面的数据分类依据的是不同的域名，数量巨大，不予表示

注：库大小和表大小均来自mongo数据库命令，库查询大小语句db.stats(1073741824);以及db.表名.stats(1073741824)

第一部分数据来源于 http://xfmro77i3lixucja.onion

第二部分数据来源

第二部分数据来源 http://fb2lib3argrtulnw.onion

涉及库 fileData_files

与第一部分类似的，第二部分数据也为一个暗网书籍杂志发布网站，

但是由于第二部分数据有部分内容大于mongo默认的单条数据大小限制（16MB）

所以第二部分采用了 mongo的GridFS存储大文件
